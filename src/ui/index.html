AsianCTV REST API

Lightweight scraper API for Asian dramas, movies & shows

v2.0 • asianctv.net
Available Endpoints
GET
/api/home
Homepage data — recent episodes, popular dramas, genres & countries
GET
/api/drama/:id
Full drama details — title, description, cast, metadata & episode list
:id = drama-slug
GET
/api/episodes/:id
Episode list only for a specific drama
:id = drama-slug
GET
/api/stream/:episodeId
Streaming links for an episode — iframe URL + all available servers
:episodeId = drama-slug-episode-N
GET
/api/search
Search for dramas or stars by keyword
?q=keyword ?type=movies|stars
GET
/api/genre/:genre
Browse dramas by genre
:genre = romance, action, comedy... ?page=1
GET
/api/country/:country
Browse dramas by country/type
:country = korean-drama, japanese-drama... ?page=1
GET
/api/list
Full alphabetical drama list
?char=a-z
